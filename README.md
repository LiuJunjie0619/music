## 这是根据一个视频教程做的vue移动端播放器
### 安装
1. 在文件夹下npm install 安装依赖包
2. 通过npm run dev启动即可

### 上线地址
#### [移动端播放器](http://47.102.220.59:5000/#/recommend)
